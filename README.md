# Sheikh Abdullah Hussain - Portfolio

Welcome to the repository of my personal portfolio website! This project showcases my skills, experience, and projects in **Web & App Development** and **Generative AI Chatbot Integration**.

## 🚀 Live Demo

[Visit My Portfolio](https://sheikhabdullah.netlify.app/)

## 🛠️ Technology Stack

This website is built with modern web technologies:

*   **Framework:** [React 19](https://react.dev/) via [Vite](https://vitejs.dev/) for fast and optimized development.
*   **Styling:** Pure CSS with CSS variables for responsive design, theming, and glassmorphism effects.
*   **Animations:** [Framer Motion](https://www.framer.com/motion/) for smooth scroll animations and element reveals.
*   **Icons:** [Lucide React](https://lucide.dev/) for crisp, customizable SVG icons.
*   **Chatbot Integration:** Google Dialogflow Messenger for a built-in AI assistant.

## ✨ Key Features

*   **Responsive Design:** Fully responsive layout with mobile-first navigation and an auto-closing menu.
*   **Dark/Light Mode:** Seamless theme switching with system preference detection (`prefers-color-scheme`) and persistence.
*   **ATS-Friendly Resume:** A dedicated `cv.html` designed with single-column layouts for 100% parsing reliability by applicant tracking systems.
*   **SEO Optimized:** Comprehensive metadata tags and custom-generated high-resolution preview images for social sharing.
*   **Premium Animations:** Scroll-triggered reveals and interactive hover states using Framer Motion.
*   **Professional Branding:** Consistent nomenclature across the portfolio and CV, with polished professional descriptions.

## 💻 Local Development

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/SheikhAbdullah1/Portfolio.git
    cd Portfolio-main
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```

4.  **Open your browser:**
    Navigate to the URL provided in the terminal (usually `http://localhost:5173`).

## 🏗️ Production

To create an optimized build, run:

```bash
npm run build
```

This will generate a production-ready `dist/` folder. You can preview the build locally using `npm run preview`.
